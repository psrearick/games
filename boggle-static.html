Boggle (Group Play) Rules

How to Use This Page

  • Choose a board size, time limit (or Untimed), and whether to show the visual timer bar, then press Start Game.
  • This screen has no word entry — everyone plays from the same board, calling out or writing down words themselves.
  • Press End Game any time to stop early. It's the only way to end an Untimed game. A timed game also ends on its own when the clock runs out.
  • When the round ends, every possible word on the board is revealed for everyone to check.

Full Rules

  • The board is a grid of lettered dice (4×4 "Classic" or 5×5 "Big"), freshly shuffled every game.
  • Build a word by chaining letters that are adjacent to each other, including diagonally. Each die can only be used once per word.
  • Words must be at least 3 letters long. "Qu" on a die counts as one letter, standing in for both Q and U (e.g. "Qu" + "I" + "T" spells QUIT).
  • This screen doesn't keep score. If you want to keep score, traditional Boggle scoring is by word length: 3–4 letters = 1 point, 5 = 2, 6 = 3, 7 = 5, 8 or more = 11 points.
